
ALL CODE IS IN THIS REPO LOOK FOR THE BarHopper FOLDER.


To install frontend, go into BarHopper/Frontend/vue

1. Install nodejs greater than 8.* and npm
2. Go here to install due command line interface, https://cli.vuejs.org/guide/installation.html
3. Run npm install (to install all dependencies)
4. Run npm run dev to start local server

To install backend, go into BarHopper/Backend/server

1. Install php 7.2
2. Install composer
3. Install Laravel by following this, https://laravel.com/docs/5.7/installation
4. run composer install to get dependencies
5. run php artisan serve to start backend
